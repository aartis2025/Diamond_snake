# AI Bio-Snake: Realistic Reptile 🐍💎

A professional, high-fidelity Snake game built with Python and Pygame. This project features an advanced AI assistant, premium glassmorphism UI, and realistic reptilian visual effects.

## Features

- **Advanced AI**: Uses BFS (Breadth-First Search) for pathfinding and a safety fallback system to avoid traps.
- **Realistic VFX**: Smooth, tapered snake body rendering with glossy reptilian textures.
- **Premium UI**: Dashboard layout with glassmorphism effects, live stat tracking, and difficulty controls.
- **Dual Modes**: Smoothly toggle between manual control and AI auto-play.
- **High Performance**: Optimized for stable FPS with varying difficulty levels (Easy, Medium, Hard).

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/USERNAME/Bio-Snake-Game.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Bio-Snake-Game
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## How to Run

Launch the game by running:
```bash
python snake_main.py
```

## Controls

- **W/A/S/D or Arrows**: Manual movement (in Human mode).
- **Space**: Toggle AI mode or Restart after Game Over.
- **Mouse**: Click buttons to change difficulty or quit.
